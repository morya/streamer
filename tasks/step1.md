# 屏幕捕获与SRT/RTMP推流工具 - 基础方案描述

## 项目概述
开发一个Windows平台下的屏幕捕获与推流工具，能够捕获屏幕指定区域，通过SRT协议（优先）或RTMP协议实时推送到网络服务。程序需具备简洁高效的快捷工具栏界面，支持动态调整捕获区域，并保持稳定的60 FPS推流性能。

## 核心功能需求
1. 屏幕捕获：基于DXcam高性能捕获库，支持60+ FPS的屏幕捕获能力
2. 区域选择：通过+/-按钮快速切换捕获区域大小（1/4屏幕、1/2屏幕、全屏）
3. 实时预览：使用红色虚线框标识当前捕获区域，窗口始终置顶
4. 视频编码：H.264硬件编码（NVENC），支持启动时设置码率
5. 网络推流：优先支持SRT低延迟协议，同时兼容RTMP协议
6. 配置持久化：在用户目录（UserDataHome）保存程序配置信息

## 目标用户
需要将屏幕内容实时推送到网络服务的技术人员、直播用户、远程协作场景

## 交付形态
完整的安装包程序（exe setup installer），用户无需安装Python环境即可使用
